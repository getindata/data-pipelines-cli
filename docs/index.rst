Installation
------------
Use the package manager `pip <https://pip.pypa.io/en/stable/>`_ to
install `dp (data-pipelines-cli) <https://pypi.org/project/data-pipelines-cli/>`_:

.. code-block:: sh

   pip install data-pipelines-cli
